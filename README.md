# Alfred SetApp Search
Search SetApp Applications via Alfred

SetApp is a paid subscription service that gives you access to 260+ Mac / iOS apps
This workflow is currently using an apps.json file that I formatted from their page: https://setapp.com/apps

The apps main features include:

- Search/filter all the apps currently listed as included
- Open page in browser about selected app
- Open DeepLink of app using UID ie: setapp://launch/12345
- SHIFT - If app is installed alfred will just open the selected app.
- SHIFT - If app Not installed show the information/install page in SetApp application.
- CMD - show the platforms this app works on
- OPT - show the Rating of the selected app
- CMD + OPT - Show both the Platforms and the ratings
